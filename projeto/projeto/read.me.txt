requisitos:
-Java 11
-banco de dados OracleXE
configuração de ambiente:
-criar conexão no banco de dados na porta 1521 com login: 'system' e senha: 'system'
-executar o 'escript de criacao banco.txt' no banco de dados
-adicionar a biblioteca ojdbc6.jar no projeto
